# Telco Customer Churn Analysis

This project analyzes telecom customer data to understand why customers leave the company (churn). The goal is to identify key factors that lead to churn and provide insights to help the telecom company improve customer retention.

---

## 📌 Project Overview

- **Industry:** Telecommunications  
- **Problem:** High customer churn impacting revenue  
- **Objective:** Identify churn patterns using customer demographics, services, contracts, and payment methods  
- **Tool Used:** Power BI Desktop  
- **Techniques:** Data cleaning (Power Query), Data modeling (DAX), Interactive dashboards  

---

## 🛠 System Approach

1. **Data Import & Preparation:**  
   - Imported telecom customer dataset into Power BI  
   - Cleaned and transformed data using Power Query  

2. **Data Modeling & Metrics:**  
   - Calculated key metrics: Total Customers, Churned Customers, Churn Rate, Average Tenure  
   - Structured the dataset for easy analysis  

3. **Dashboard & Visualization:**  
   - Created multiple pages: Summary, Customer Demographics, Service Analysis, Contract & Payment Analysis  
   - Used visuals: KPI cards, Pie charts, Bar/Column charts, Line & Clustered Column charts, Decomposition Trees  

4. **Interactive Features:**  
   - Page navigation tabs, slicers, and Reset Filters button  

5. **Insights & Results:**  
   - Identified high-risk churn segments: Month-to-month contracts, Fiber Optic users, lack of technical support or online security  
   - Insights can guide targeted retention strategies  

---

## 📊 Key Results

- **Churn Rate:** 25% of customers left the company  
- **High-Risk Segments:** Month-to-month contracts, Fiber Optic users, customers without support services  
- **Demographics Insights:** Balanced gender distribution, fewer senior citizens, average tenure ~34 months  
- **Contract & Payment Patterns:** Some payment methods and higher monthly charges linked to higher churn
